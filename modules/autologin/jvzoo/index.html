Add/Remove ActiveCampaign Tag/Tags, Set Contact Field When Achievement Earned

Important

Be aware that these ActiveCampaign actions are only applied when the achievement is earned. If the achievement is revoked in any way the actions are not reversed.

  1. When editing an individual achievement within the Achievement Data panel in the ActiveMember360 Add Tags dropdown select the ActiveCampaign tag(s) to be added when the achievement is earned.
  2. In the ActiveMember360 Remove Tags dropdown select the ActiveCampaign tag(s) to be removed when the achievement is earned.
    Note

    Remove tags are processed before add tags to ensure any ActiveCampaign automations can be triggered appropriately.

  3. In the ActiveMember360 Set Contact Field dropdown select the ActiveCampaign field to set to the value specified in step 4.
    Note

    Only ActiveCampaign text and text area fields are supported and shown for selection.

  4. Specify the value to be set for the contact field.
  5. Ensure to click Update to save the achievement changes.
