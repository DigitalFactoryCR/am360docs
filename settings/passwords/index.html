Passwords

Store encrypted passwords in ActiveCampaign#

Select if you wish to store encrypted passwords in ActiveCampaign.

Beware

Switching the password encryption mode to YES is irreversible.

Before changing this setting to YES please ensure you have considered the implications of using encrypted passwords.

Please refer to Passwords.

Default: No

Minimum password characters#

Set the minimum number of characters required for a password generated by ActiveMember360.

Default: 8

Maximum password characters#

Set the maximum number of characters required for a password generated by ActiveMember360.

Default: 16

Allowed character set#

Define the character set used to generate a password.

Default: abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ0123456789!@#\$%^*()
