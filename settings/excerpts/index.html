Excerpts

Force excerpts at all times#

Activating this option will automatically create an excerpt for pages and posts that do not have one.

It will use the content of the page/post to generate the excerpt.

Beware

Please take particular care when using this option with page builders or content containing shortcodes.

You should check you do not expose more content in the excerpt than desired.

In all cases it is best to use a manual excerpt for each page/post.

Default: No

Enable blur effect for excerpts#

Set to Yes if you wish to show 'blurred' content after the excerpt and any common excerpt text.

Only 'blurred' text content will be shown, not 'blurred' images.

When this option is enabled you can also show the page/post excerpt with a gradient.

Default: No

Excerpt gradient start color#

Specify the start color for the excerpt gradient.

If no gradient is required when Enable blur effect for excerpts is enabled set both gradient start and end color to the same value.

This color setting overrides that of your theme CSS or similar.

Default: #000000

(This setting is only available if MembersPlus, Settings, Excerpts, Show excerpt with gradient is set to Yes.)

Excerpt gradient end color#

Specify the end color for the excerpt gradient.

If no gradient is required when Enable blur effect for excerpts is enabled set both gradient start and end color to the same value.

This color setting overrides that of your theme CSS or similar.

Default: #ffffff

Disable excerpt links#

Activating this option will automatically show a link at the end of the excerpt that when clicked will show the access level denied info popup defined in MembersPlus, Settings, Access Levels if the user has the access level denied status when viewing the excerpt.

The link will not show for any visitor that does not have the access level denied status when viewing the excerpt.

The purpose of this option is so that users with the access level denied status can click the link for an explanation of why they have been denied access to the page/post.

Default: No

Max. length for WP-generated excerpts#

In various locations in your WordPress site, such as an archive or category view of multiple posts, WordPress will automatically generate an excerpt if one hasnâ€™t be defined.

By default, WordPress creates excerpts using the first 55 words of a page/post.

By modifying this value, you can shorten or lengthen the length of your auto-generated excerpts to better suit your requirements.

This setting only applies if Force excerpts at all times is set to Yes.

Default: 55

Common excerpt text#

Whilst you can define individual excerpts for each page/post there may be some content you wish to show in every excerpt.

The content defined here will be automatically appended to the individual excerpt for each page/post.

It will be displayed after the individual excerpt but before 'blurred' content if enabled.

Default: This is an excerpt. You do not have sufficient rights to view the full content.
