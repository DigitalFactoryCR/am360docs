Mail Options

In various situations, ActiveMember360 interacts with your members through email sent from WordPress (e.g. password send).

Default "Mail From" name#

Here you can specify an alternate email 'From:' name rather than using the default of 'Site Admin'.

Default: Site Admin

Default "Mail From" address#

Here you can specify an alternate email 'From:' address rather than using the WordPress default.

Default: As set in WordPress, Settings, General, Administration Email Address

Note

We would always recommend using an email domain that matches your WordPress domain to minimise email delivery issues or problems if using SMTP.
