Amazon S3

S3 access key.#

enter your S3 access key as provided by Amazon.

S3 secret key.#

enter your S3 secret key as provided by Amazon.

Default expiration in seconds#

enter the number of seconds after which any generated link should expire.The default expiration for all your expiring links. Can be overriden in each single shortcode.

Default S3 host#

Enter the full CNAME to your S3 host, excluding http:// or https://. Your default S3 host. Can be overriden in each single shortcode.

Default S3 bucket#

enter the name of your default S3 bucket. This may be overriden in any shortcode by using the "bucket" parameter. Your default S3 bucket. Can be overriden in each single shortcode.
