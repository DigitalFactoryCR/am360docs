Creating WordPress Users

WordPress users can be created in many ways; using only WordPress, MembersPlus or by other third party plugins.

MembersPlus provides some alternative methods to create a WordPress user. However these all use the standard WordPress user and password creation processes.

MemberPlus does not control in any way the generation of the WordPress password. So, the length, characters permitted for the password, and password strength are all controlled directly by WordPress.

The main options for creating a WordPress user are: Using the MembersPlus webhook â€“ Add a WordPress User

It is most likely that you do not want to add a WordPress user until that person subscribes to an email list, requests a lead magnet, or purchases a product or subscription from you.

MembersPlus provides a webhook that can be triggered by a HTTP POST sent from your primary CRM to your WordPress site running MembersPlus that will create a WordPress user.

This is the MembersPlus Add a WordPress User webhook.

This webhook can be used multiple times to create WordPress users on any MembersPlus powered site once the contact exists within the primary CRM.

The user will be created with a username matching that of their email address and the WordPress role as specified in WordPress, Settings, General, New User Default Role. If the MembersPlus, Module, User Registration is enabled the settings there will also apply, to add specific primary CRM tags and set a contact field.

Details of how to implement this webhook are available at MembersPlus, Webhooks, Add a WordPress User. Using the MembersPlus module â€“ Auto-Login

MembersPlus provides a auto-login capability that can be used to automatically login a visitor to your WordPress MembersPlus site.

This can be used to autologin a user; after a primary CRM form submission, between different WordPress MembersPlus sites or if you are using an external cart (i.e. one not hosted in WordPress) MembersPlus provides a module that allows validated purchasers to autologin to the WordPress MembersPlus site using an autologin URL as the Success or Thank You Page URL for the purchase.

This is the MembersPlus Auto-Login module.

The Auto-Login module when used in conjunction with an external cart such as; ThriveCart, Digistore24, ClickBank or JVZoo will automatically create a contact in if the order credentials passed to MembersPlus validate successfully and the corresponding user does not already exist.

The user will be created with a username matching that of their email address and the WordPress role as specified in WordPress, Settings, General, New User Default Role.

Also when using this module with a contact in your primary CRM if the corresponding contact, i.e. a contact with a matching email address, does not already exist.

If the MembersPlus, Module, User Registration is enabled the settings there will also apply, to add specific primary CRM tags and set a contact field.

Details of how to implement this webhook are available at MembersPlus, Modules, Auto-Login. Using the MembersPlus shortcode â€“ [mbr_form_register]

MembersPlus provides the shortcode [mbr_form_register] which can be used on any post/page to allow a visitor to register on the WordPress site.

The user will be created with a username matching that of their email address, and the WordPress role as specified in WordPress, Settings, General, New User Default Role.

When using this shortcode the user is always created as a contact in your primary CRM if the corresponding contact, i.e. a contact with a matching email address, does not already exist.

If the MembersPlus, Module, User Registration is enabled the settings there will also apply, to add specific primary CRM tags and set a contact field.

Details of how to use this form are available at MembersPlus Shortcode [mbr_form_register]. WordPress Administrator User Functions

A WordPress Administrator can manually create WordPress users using the WordPress administrator function WordPress, Users, Add New.

When users are created using this method they can be added as contacts to, or updated within, the primary CRM with the addition of specified tags if the MembersPlus, Module, User Registration is enabled and the settings there apply. WordPress Registration

WordPress comes with built-in functionality to allow site visitors to register as WordPress users.

This is controlled by the setting WordPress, Settings, General, Anyone can register.

When this is enabled there will be a register link shown on the standard WordPress login page.

When users are created using this method they can also be added as contacts to, or updated within, the primary CRM with the addition of specified tags and the setting of a contact field if the MembersPlus, Module, User Registration is enabled and the settings there apply. Third Party WordPress Plugins

Many other third party WordPress plugins also include features to create WordPress users.

Any of these can be used to create your WordPress user.

If the user is created using standard WordPress functions they can be added as contacts to, or updated within, the primary CRM with the addition of specified tags and the setting of a contact field if the MembersPlus, Module, User Registration is enabled and the settings there apply. WooCommerce

A notable example of a third party WordPress plugin that can create WordPress user is WooCommerce.

When using WooCommerce users can be created in WordPress using the WooCommerce, Settings, Accounts & Privacy, Account Creation features.

Users created by WooCommerce will be given by default the custom WordPress user role of Customer.

When users are created using this method they can be added as contacts to, or updated within, the primary CRM with the addition of specified tags and the setting of a contact field if the MembersPlus, Module, User Registration is enabled and the settings there apply.

Also, if using the MembersPlus WooCommerce Extension that can also control the creation or update of a contact within the primary CRM with the addition of specified tags etc dependent upon the visitors actions within WooCommerce. Import

There may be situations where you wish to bulk create users in your WordPress site.

MembersPlus does not provide a facility to bulk create users.

However, there are many third party plugins that can do this for you. Typically, these work with a CSV (Comma Separated Values) file.

An example of such a plugin is Import Export WordPress Users. href="https://wordpress.org/plugins/users-customers-import-export-for-wp-woocommerce/">Import Export WordPress Users
