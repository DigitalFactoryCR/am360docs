WordPress Login

ActiveMember360 treats certain pages within your WordPress site as special pages.

These pages will be accessible to any visitor to the site irrespective of:

  • Their visitor type/status.
  • The ActiveMember360 Protection Settings in place for the individual pages/posts.

When ActiveMember360, Plugin Options, Login Settings, Always require login is active the following are treated as special pages:

  • Login page selected in ActiveMember360, Plugin Options, Special Pages, Select your login page.
  • Register page selected in ActiveMember360, Plugin Options, Special Pages, Select your registration page.

When ActiveMember360, Plugin Options, Login Settings, Always require login is not active the following are treated as special pages:

  • Site home page.
  • Login page selected in ActiveMember360, Plugin Options, Special Pages, Select your login page.
  • Register page selected in ActiveMember360, Plugin Options, Special Pages, Select your registration page.
  • Alternate 404 page selected in ActiveMember360, Plugin Options, Special Pages, Alternate "404 - Page not found" page.

Similarly, WordPress wp-login.php and theme template 404.php will always be accessible when using ActiveMember360.

Beware

Please be aware that there are many other third party plugins that may override the ActiveMember360 behaviour for wp-login.php and 404.php.
